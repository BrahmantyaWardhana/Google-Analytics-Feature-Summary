# Google-Analytics-Feature-Summary

Summarized documentation for Google Analytics features

All information here is taken from Google Analytics'
[documentation](https://support.google.com/analytics/#topic=).

# Data Reports

All data are shown in greater detail in the
[Reports tab](https://analytics.google.com/analytics/web/?authuser=1#/a206625290p285025533/reports/start?params=_u..nav%3Dmaui&collectionId=10831543755).

## Life Cycle

### 1. Acquisition

Shows how users find the app:

- User Acquisition = Where new users come from.
- Traffic Acquisition = Where all sessions come from.
- Non Google Campaign = Campaign traffic that did _not_ originate from Google-owned sources.
- User Acquisition Cohort = How newly acquired users behave over time within the selected period.
- Lead Acquisition = Shows which channels and campaigns generate leads via events like `generate_lead`, `submit_form`, etc. ([documentation](https://support.google.com/analytics/answer/16376749))

---

### 2. Engagement

Shows what users do after they arrive:

- Events = User actions tracked as events (page views, clicks, scrolls, downloads).
- Conversions = Events marked as key actions or goals (e.g., generate_lead, purchase).
- Pages & Screens = Which pages or screens users view the most.
- Landing Pages = Pages where users start their sessions.
- User Interaction = How long users stay engaged on the app.

---

### 3. Monetization

Shows revenue-related user actions (only appears if monetization tracking is enabled):

- Purchases = Ecommerce purchase events and transaction details.
- Revenue = Total revenue generated.
- In-App Purchases = Purchases made inside mobile apps.
- Ads Monetization = Estimated earnings from ad impressions.

---

### 4. Retention

Shows how often users return after their first visit:

- New vs Returning Users = Differentiates first-time users from returning visitors.
- User Retention = Percentage of users who return after Day 1, Day 7, Day 30, etc.
- Cohorts = Groups of users acquired during the same period.
- Lifetime Value Trends = Long-term engagement and value generated by cohorts.

---

## User

### 1. User Attributes

Shows characteristics of your users:

- Demographic Detail = Breaks down users by age, gender, location, and interests (when Google Signals is enabled).
- Audiences = Groups of users defined by conditions (e.g., “Returning Users,” “High-Value Users,” “Engaged Users”). Useful for analysis and remarketing.

---

### 2. Tech Details

Shows the technologies users use to access the app:

- Tech Details = Device type, operating system, browser, screen resolution, app version, platform (iOS/Android/Web), and device category (mobile/desktop/tablet).
